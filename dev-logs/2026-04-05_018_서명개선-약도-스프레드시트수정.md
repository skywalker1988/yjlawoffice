## 프롬프트
> 자필 서명 기능이 너무 허접하다. 마우스도 제대로 인식 못해. 최고급 자필 서명 기능 구현시켜줄 것.
> 교대역에서 오시는 길 부분 그림 현재 주소 반영해서 다시 그려라.
> 스프레드시트 1행에 관련 내용도 추가해줘야지?
> Apps Script 새 배포 URL 반영

## 작업 내용

### 1. 서명 패드 고급화
- 속도 기반 필압 변환: 빠르게 쓰면 가늘게, 느리게 쓰면 굵게
- 베지어 곡선(quadraticCurveTo)으로 부드러운 획
- 캔버스 해상도 2배 (680x240 → 실제 표시 100%x140px)로 선명도 향상
- 빈 캔버스 체크 — 아무것도 안 쓰면 서명 완료 불가
- DPI 스케일 보정으로 마우스/터치 좌표 정확도 향상

### 2. 약도 SVG 새 주소 반영 (directions-map.svg)
- "오퓨런스빌딩 7F" → "서초대로 327, 5F"
- 사무소 위치를 서초대로 327 위치에 맞게 이동 (교대역과 서초역 사이)
- 도보 시간: 서초역 3분→5분, 교대역 8분→7분
- 타이틀: "교대역 → 윤정 법률사무소 도보 약도" → "오시는 길 안내"

### 3. Apps Script 업데이트
- 새 배포 URL 반영 (개인 Gmail 계정으로 재배포)
- 스프레드시트 헤더: 10개 컬럼 (접수일시, 이름, 전화번호, 이메일, 상담분야, 희망날짜, 희망시간, 상담내용, 개인정보동의, 상태)
- 기존 테스트 데이터 삭제, 헤더만 유지

### 4. 백엔드 DB/웹훅 분리
- DB에 없는 필드(preferredDate, preferredTime, agreed)가 insert 에러를 일으키는 문제 수정
- dbData(DB용)와 webhookData(Apps Script용) 분리
