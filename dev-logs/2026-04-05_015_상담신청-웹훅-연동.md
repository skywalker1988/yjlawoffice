# 2026-04-05 | 온라인 상담 신청 → 이메일 + 스프레드시트 연동 구현

## 프롬프트
> 온라인 상담 신청을 하면 관련 내용이 내 이메일로도 오고, 구글 스프레드 시트에도 저장되었으면 좋겠어
> (Google Cloud 서비스 계정 권한 문제로 방법2 Apps Script로 전환)

## 최종 구현 방식
- **Google Apps Script 웹훅** 방식 채택
- Apps Script가 이메일 발송 + 스프레드시트 저장 둘 다 처리
- 백엔드에서 상담 신청 시 비동기로 웹훅 호출 (실패해도 DB 저장은 정상 처리)

## 구성

### Apps Script (Google 스프레드시트 내)
- 스프레드시트: https://docs.google.com/spreadsheets/d/1wDjbjkuWr2nGD1BsypTF9goQy6tdduAREW7euNSQqVI
- 웹 앱 URL: https://script.google.com/a/macros/younjeong.com/s/AKfycbyFjsUCkBCHS33ykbYfBvTLd2yn8QkAXe0QVfY0i8Hw9V0ZaSTfFV7mTWId0mLC7VHO/exec
- 기능: doPost()로 JSON 수신 → 스프레드시트 행 추가 + 이메일 발송
- 이메일 수신: younsehwan@younjeong.com

### 백엔드 (`sb-consultations.js`)
- POST /api/sb/consultations 핸들러에 `notifyAppsScript()` 추가
- DB 저장 후 비동기로 Apps Script 웹훅 호출
- 카테고리 코드를 한국어 라벨로 변환하여 전송
