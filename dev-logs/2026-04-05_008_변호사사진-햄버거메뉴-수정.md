# 2026-04-05 | 변호사 사진 비율 + 햄버거 메뉴 배경색 수정

## 프롬프트
> 김수경 변호사랑 김효림 변호사는 얼굴이 잘려서 나오잖아. 비율 적절하게 조절해라.
> 햄버거 메뉴 클릭했을때 배경이 아예 흰색으로 나오는데 적절하게 색 변경해봐

## 문제
1. **변호사 카드 사진**: `center/cover`로 이미지 중앙 기준 크롭 → 인물 얼굴이 잘림
2. **햄버거 메뉴**: 풀스크린 메뉴 배경이 `rgba(255,255,255,0.97)` 순백색 → 사이트 다크 톤과 불일치

## 작업 내용

### 1. 변호사 사진 크롭 수정 (`LawyersPage.jsx`)
- 카드 사진 `background-position`을 `center` → `center top`으로 변경
- 얼굴이 상단에 위치하는 인물 사진 특성에 맞게 상단 기준 크롭

### 2. 햄버거 메뉴 배경색 변경 (`Layout.jsx`)
- 메뉴 배경: `rgba(255,255,255,0.97)` → `rgba(15,25,35,0.97)` (다크 네이비)
- 메뉴 텍스트: `#ccc` → `rgba(255,255,255,0.5)` (다크 배경에 맞는 밝은 톤)
- 햄버거 아이콘: 메뉴 오픈 시 흰색으로 전환 (`menuOpen` 상태 반영)
- 관리자 링크 border 색상 다크 배경에 맞게 조정

## 수정 파일
- `frontend/src/pages/LawyersPage.jsx` — 사진 position 수정
- `frontend/src/components/Layout.jsx` — 메뉴 배경/텍스트/아이콘 색상 수정
