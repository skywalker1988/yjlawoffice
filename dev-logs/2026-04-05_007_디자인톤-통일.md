# 2026-04-05 | 디자인 톤 통일 — 다크 네이비 + 골드 프리미엄 로펌 스타일

## 프롬프트
> 사무실 홈페이지 랜딩페이지가 현재 맨하탄 스카이라인 동영상인데 이것과 비슷한 톤으로 사무실 홈페이지 색상, 디자인톤도 가야되는거 아닐까? 너무 중구난방인데

## 문제
- `--accent-gold`가 실제로는 `#6366f1` (보라색/인디고) — 골드가 아니었음
- 서브페이지 히어로 배경색, 푸터, 관리자 페이지 악센트 색상이 제각각
- 맨하탄 스카이라인의 프리미엄 다크 톤과 불일치

## 디자인 토큰 변경

| 항목 | 변경 전 | 변경 후 |
|------|---------|---------|
| accent-gold | #6366f1 (인디고) | #b08d57 (진짜 골드) |
| accent-gold-hover | #4f46e5 | #9a7a48 |
| bg-primary | #ffffff | #faf9f7 (워밍 화이트) |
| bg-secondary | #f5f5f3 | #f3f1ed |
| bg-dark | #1a1a1a | #0f1923 (딥 네이비) |
| 신규: bg-navy | — | #1a1a2e |
| 신규: accent-gold-light | — | rgba(176,141,87,0.12) |

## 적용 범위
- **CSS 토큰** (index.css): 전체 색상 체계 변경
- **서브페이지 히어로**: 모두 `#0f1923 → #1a2332` 다크 네이비 그라디언트로 통일
- **푸터**: 밝은 배경 → 다크 네이비(#0f1923) + 골드 텍스트
- **관리자 레이아웃**: 인디고 악센트 → 골드 악센트
- **변호사 카드 기본 배경**: 다크 네이비 통일
- **지도 탭 활성 색상**: 다크 네이비
