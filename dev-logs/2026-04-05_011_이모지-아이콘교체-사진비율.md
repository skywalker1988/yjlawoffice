# 2026-04-05 | 이모지 → Lucide 아이콘 교체 + 변호사 사진 비율 수정

## 프롬프트
> 랜딩 페이지를 비롯해서 전체적으로 홈페이지 사용되고 있는 이모티콘이 너무 조잡한거 같아. 중학생 작품같아. 프로페셔널한 모노톤 아이콘 찾아서 반영해줄것. 그리고 15년 이상 실무경험 없으니까 빼주고.
> 김수경 변호사님이랑 김효림 변호사님 사진 여전히 너무 얼굴이 꽉차보여.
> 변호사 이름을 클릭하면 팝업 형식으로 뜨는건 좋은데 사진이 잘려서 나오잖아.

## 작업 내용

### 1. 이모지 → Lucide React 아이콘 교체 (4개 페이지, 19곳)

**HomePage.jsx**
- "15+" 통계 항목 삭제, 그리드 4열 → 3열
- Users, Clock, ShieldCheck 아이콘 (골드 모노톤)

**AboutPage.jsx**
- Handshake(신뢰), Scale(전문성), Landmark(헌신), Lightbulb(혁신)

**PracticePage.jsx** (11개 업무분야)
- Scale, ShieldAlert, Heart, Building2, Landmark, Receipt, Home, FileText, Mail, Handshake, Briefcase

**ConsultationPage.jsx**
- Phone, MessageCircle, AtSign, Clock

**LawyersPage.jsx**
- Scale(플레이스홀더), MailIcon, PhoneIcon

### 2. 변호사 사진 비율 수정
- 카드: background-image → `<img>` 태그로 변경, `aspect-ratio: 3/4`, `object-position: center 20%`
- 모달: `aspect-ratio: 4/3`, `object-position: center 20%`
- 얼굴이 위쪽에 위치하므로 20% 지점에 포커스

### 아이콘 스타일 통일
- 색상: `#b08d57` (골드 악센트)
- 선 굵기: `strokeWidth={1.3}` (얇고 세련된 느낌)
- 크기: 26~32px (용도에 따라)
