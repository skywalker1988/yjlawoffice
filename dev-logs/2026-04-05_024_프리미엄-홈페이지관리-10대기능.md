## 구현된 10대 기능

### Site Manager 내 탭 (7→10개)
1. **실시간 미리보기** — 분할 화면(55/45%), iframe + postMessage, 데스크톱/모바일 전환
2. **변경 이력 + 되돌리기** — site_settings_history 테이블, 타임라인 UI, 원클릭 복원, 기본값 초기화
3. **SEO 관리** — 페이지별 meta title/description/OG image/keywords, 구글 검색 미리보기
4. **공지/배너/팝업 관리** — announcements 테이블, CRUD, 시작/종료일 스케줄, 컬러 피커
5. **섹션 순서 + 표시/숨기기** — 드래그앤드롭 레이아웃 편집, 섹션 가시성 토글
6. **다국어 (한/영)** — useSiteSettings 언어 지원, 각 탭에 한/영 토글, Layout 언어 전환 버튼
7. **예약 발행** — scheduled_changes 테이블, scheduler.js (60초 간격 체크), 자동 적용

### 별도 관리자 페이지
8. **미디어 관리자** (`/admin/media`) — 드래그앤드롭 업로드, 폴더 관리, 파일 미리보기, MediaPicker 모달
9. **접근 권한 관리** (`/admin/settings`) — admin_users 테이블, 역할(admin/editor/viewer), crypto.scrypt 해싱, 세션 관리, 활동 로그, 개발 이력
10. **분석 대시보드** (`/admin/analytics`) — page_views 테이블, 미들웨어 자동 로깅, Canvas 차트(꺾은선/막대), CSV 내보내기

### 추가 기능
- **개발 역사 확인** — dev-logs API + 관리자 설정 페이지에서 조회 가능
- **공개 배너/팝업** — Announcements.jsx 컴포넌트, Layout에 자동 표시
