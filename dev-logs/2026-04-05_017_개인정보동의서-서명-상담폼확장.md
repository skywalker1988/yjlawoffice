# 2026-04-05 | 개인정보 동의서 모달 + 자필 서명 + 상담 폼 확장

## 프롬프트
> 개인정보보호법령에 근거해서 의뢰인이 상담신청 전에 개인정보보호 동의 클릭하려고 하면 관련 동의서 읽을 수 있게 만들어줘. 반드시 끝까지 스크롤 다하고 '네 확인했습니다' 클릭해야 상담신청할 수 있게 만들어줘.
> 자기 이름 석자 자필서명할 수 있도록 기능구현해주자. 아이폰이나 갤럭시 등 모바일에서도 터치해서 서명가능하게.
> 개인정보보호 관련 내용 동의 했는지도 체크해서 승낙여부 같이 엑셀로 받아보고 싶은데
> 언제 상담을 희망하는지도 날짜랑 가능한 시간대도

## 작업 내용

### 1. 상담 폼 필드 추가
- 희망 상담 날짜 (date picker, 오늘 이후만 선택 가능)
- 희망 시간대 (9시~18시, 1시간 단위 드롭다운)

### 2. 개인정보 동의서 모달
- 「개인정보 보호법」 제15조, 제17조, 제22조 근거 동의서 작성
- 내용: 수집목적, 수집항목, 보유기간, 제3자 제공, 파기 절차, 정보주체 권리, 보호책임자, 동의 거부 권리
- 끝까지 스크롤해야 서명 패드 + 동의 버튼 활성화

### 3. 전자 서명 패드
- Canvas 기반 서명 패드
- 마우스(PC) + 터치(모바일) 모두 지원 (`touchAction: "none"`)
- "다시 쓰기" / "서명 완료" 버튼
- 서명 완료 후에만 "네, 확인했습니다" 버튼 활성화

### 4. 스프레드시트/이메일 확장
- 개인정보 동의 여부 ("동의"/"미동의") 컬럼 추가
- 희망 상담 날짜, 시간대 컬럼 추가
