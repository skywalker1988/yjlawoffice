# 2026-04-05 | 고객 DB 자동 등록 + 메시지 발송 연동

## 프롬프트
> 상담신청을 하면 자동으로 고객 DB에 추가를 해줘. 그리고 메시지 발송할때도 선택할 수 있게 만들어주고.
> 상담신청할때 이메일이든 전화번호든 둘 중 하나는 꼭 입력해야 상담신청이 가능하게 기능 구현해줘

## 구현 내용

### 1. 고객 DB (clients 테이블)
- id, name, phone, email, category, memo, source, consultationId, isActive
- source: "consultation"(상담 자동), "referral"(소개), "manual"(직접), "other"
- 전화번호 기준 중복 체크 (이메일로도 보조 검색)

### 2. 상담 신청 → 고객 자동 등록
- `sb-consultations.js` POST 핸들러에서 상담 접수 후 자동으로 clients 테이블에 upsert
- 기존 고객(전화번호/이메일 일치)이면 정보 업데이트
- 신규 고객이면 source="consultation"으로 자동 생성

### 3. 상담 신청 시 연락처 검증 변경
- **변경 전**: 전화번호 필수, 이메일 선택
- **변경 후**: 전화번호 또는 이메일 중 최소 1개 필수
- 프론트엔드(ConsultationPage.jsx) + 백엔드(sb-consultations.js) 모두 검증

### 4. 메시지 발송 수신자 소스 선택
- SendTab에 "고객 DB" / "상담 신청" 소스 전환 버튼 추가
- 고객 DB 선택 시 `/api/sb/clients`에서 로드
- 상담 신청 선택 시 기존 `/api/sb/consultations`에서 로드

### 5. 관리자 고객 관리 페이지 (AdminClients.jsx)
- 고객 CRUD (등록/수정/삭제/활성·비활성)
- 이름/전화번호/이메일 검색
- 출처 뱃지 (���담/소개/직접)
- 페이지네이션
