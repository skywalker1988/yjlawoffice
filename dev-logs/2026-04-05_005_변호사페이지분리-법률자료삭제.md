# 2026-04-05 | 변호사 소개 별도 페이지 분리 + 법률자료 메뉴 삭제

## 프롬프트
> 변호사 소개 페이지가 안보이는데? 어디다 숨겨둔거냐. 그리고 법률자료 메뉴도 없애라. 관련된 데이터도 다

## 작업 내용

### 1. 변호사 소개 별도 페이지로 분리
- `LawyersPage.jsx` 신규 생성 — 별도 히어로 + 카드 그리드 + 상세 모달
- `AboutPage.jsx`에서 변호사 섹션 제거 (핵심 가치 + 연혁만 유지)
- 네비게이션에 "변호사 소개" (`/lawyers`) 메뉴 추가
- 라우터에 `/lawyers` 경로 등록

### 2. 법률자료 메뉴 및 관련 라우트 삭제
- 네비게이션에서 "법률자료" 제거
- 라우터에서 `/vault`, `/vault/:id`, `/graph`, `/timeline` 제거
- VaultPage, DocumentDetailPage, GraphPage, TimelinePage import 제거

### 3. 홈페이지 리뉴얼
- 검색바 제거 (법률사무소에 불필요)
- 대시보드 통계/문서 목록 제거
- 법률사무소 메인 랜딩으로 변경: 핵심 지표, 접근 방식, CTA
- "상담 신청" + "업무분야 보기" 듀얼 CTA 버튼

### 현재 네비게이션 구조
1. 사무소 소개 → /about
2. 업무분야 → /practice
3. **변호사 소개 → /lawyers** (신규 분리)
4. 상담안내 → /consultation
