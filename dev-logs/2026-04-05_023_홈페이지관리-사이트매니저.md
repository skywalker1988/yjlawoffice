# 2026-04-05 | 홈페이지 관리 (Site Manager) 관리자 페이지

## 프롬프트
> 관리자 페이지에서 홈페이지 관리를 할 수 있는 기능을 별도로 구현해줘. 거기서 홈페이지 전체 테마, 디자인, 레이아웃, 각 섹션별 문구 전부 수정할 수 있게 해줘. 직원들도 여기 들어와서 쉽게 편집할 수 있도록 최고급으로 기능 구현해주고 직관적으로 만들어줘. 히어로 영상도 여기 안에 포함시켜줘.

## 구현 내용

### 핵심 설계
- **site_settings** DB 테이블: page + section 키로 JSON 콘텐츠 저장
- DB에 데이터 없으면 현재 하드코딩된 값 그대로 표시 (제로 설정)
- 기존 히어로 영상 관리 컴포넌트를 탭 안에 임베드

### AdminSiteManager — 6탭 구성
1. **홈페이지**: 히어로(제목/부제/슬로건/CTA), 통계 3개, 접근방식, 하이라이트 3개, 하단 CTA
2. **소개**: 히어로, 철학, 핵심가치 4개, 연혁 타임라인
3. **업무분야**: 히어로, 소개글, 업무분야 11개(아코디언 편집기)
4. **공통(헤더/푸터)**: 네비 메뉴 6개, 푸터 정보
5. **히어로 영상**: 기존 AdminHeroVideos 컴포넌트 임베드
6. **테마**: 악센트 컬러, 텍스트 컬러, 배경색 등 (컬러 피커)

### 주요 기능
- 반복 항목 추가/삭제 (통계, 하이라이트, 핵심가치, 연혁, 업무분야)
- 업무분야 아코디언 패턴 (펼치기/접기)
- 변경사항 감지 → 하단 고정 저장 바
- 일괄 저장 (bulk upsert 트랜잭션)
- "저장되었습니다" 토스트 메시지

### 공개 페이지 동적 설정 적용
- useSiteSettingsPage 훅: 페이지별 설정 로드, 모듈 레벨 캐시
- HomePage, AboutPage, PracticePage, Layout 4개 파일 수정
- 아이콘(Lucide)은 원본 배열에서, 텍스트는 DB 설정에서 로드
